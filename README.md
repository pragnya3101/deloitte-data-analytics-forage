# Deloitte Australia Data Analytics Virtual Experience (Forage)

## Overview
This repository contains my work completed as part of the Deloitte Australia Data Analytics Virtual Experience Program on Forage.

## Project Tasks

### Task 1: Data Analysis
- Imported telemetry data into Tableau.
- Created a calculated field to measure machine downtime.
- Built an interactive dashboard showing:
  - Down Time per Factory
  - Down Time per Device Type
- Applied dashboard filters to analyze downtime by factory.

### Task 2: Forensic Technology
- Classified employee equality scores using Microsoft Excel.
- Used Excel formulas to categorize records into:
  - Fair
  - Unfair
  - Highly Discriminative
